# Bike Sales Dashboard

## Introduction

This Bike Sales Dashboard was created to analyze customer purchasing behavior and identify factors that influence bike sales.

The dashboard provides interactive visualizations of customer demographics and purchasing patterns, allowing users to explore trends based on different filters.

### Dashboard File

My final dashboard is in [Bike_Sales_Dashboard.xlsx](Bike_Sales_Dashboard.xlsx).

---

## Excel Skills Used

The following Excel skills were utilized for this project:

- **Pivot Tables**
- **Pivot Charts**
- **Slicers**
- **Charts**
- **Formulas and Functions**
- **Dashboard Design**

---

## Dataset

The dataset contains customer information related to bike purchases. It includes details such as:

- **Gender**
- **Marital Status**
- **Education Level**
- **Region**
- **Age Brackets**
- **Income**
- **Commute Distance**
- **Purchased Bike Status**

---

## Dashboard Features

The dashboard allows users to:

- Analyze average income by gender and purchase status.
- Explore bike purchases across different age groups.
- Examine the relationship between commute distance and bike purchases.
- Filter results by marital status, region, and education level using interactive slicers.

---

## Conclusion

This project demonstrates how Excel can be used to transform raw customer data into an interactive dashboard. The dashboard provides insights into customer demographics and purchasing behavior, helping users better understand the factors associated with bike sales.
